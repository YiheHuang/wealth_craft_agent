## 课程主线

整套课程不是单纯的“图形识别术”，而是一套把市场认知、风险控制、结构分解、买卖点、级别联立和操作纪律连接起来的方法论。

从文章脉络看，可以大致分成六条主线：
1. 入门观念：市场没有救世主，只有赢家和输家，交易首先是生存问题。
2. 走势语言：趋势、盘整、背驰、中枢、走势类型、同级别分解，是描述市场的基础语法。
3. 结构几何：分型、笔、线段、中枢、线段破坏、区间套，是把K线抽象成结构的核心工具。
4. 操作系统：三类买卖点、节奏、当下、持股持币、区间套与多级别联动，决定怎么下手。
5. 风险控制：资金管理、程序组独立性、政策风险、消息扰动、机械执行，决定怎么活下来。
6. 认知校正：杜绝一根筋思维、承认走势多义性、承认结构未完成，是避免“伪缠论”的关键。

## Agent B 使用原则

当用户要求“用缠论分析最近三个月日K线图”时，Agent B 必须遵守以下规则：
1. 先锁定级别和区间：这里默认是最近三个月日K，不要偷换成月线大势总评。
2. 先描述数据，再下缠论结论：必须先交代区间起止、阶段高低点、涨跌幅、关键波段。
3. 先结构，后结论：先分型、笔、线段，再中枢，再背驰，再买卖点。
4. 能确认的说确认，不能确认的说候选：结构未走完时不能硬判。
5. 追问场景只回答当前主题：用户只问日K缠论，就不要把新闻、财务、主营业务重新总结一遍。
6. 禁止泛泛谈概念：必须尽量落到具体日期、价位、最近高低点与结构变化。

核心结论：
- 市场不是道德场，而是收益与风险的现实场。
- 进入市场后，唯一可检验标准是能否稳定盈利。
- 交易系统不能依赖单一神奇信号，必须依赖可执行的程序、资金管理和纪律。
- 任何“我喜欢这只票”“这公司我熟”“别人说很好”的理由，在缠论框架里都不是买入依据。
- 走势分析最终要落回“看当下、按结构行动”，不是沉迷叙事。

对 Agent B 的含义：
- 在做 K 线分析时，不要把主观偏好写成结论。
- 任何缠论结论都必须附带不确定性与触发条件。
- 若用户只问局部走势，回答也应聚焦“当下结构是否支持动作”，而不是长篇宏大叙事。

核心结论：
- 没有趋势，就没有背驰；背驰必须依附于明确走势推进。
- 走势不是杂乱K线堆，而是可被分解、可被连接、可被层级化的结构。
- “走势终完美”不是说你一定能预测，而是说走势总会走成某种可描述结构。
- 同级别分解与走势类型连接，是判断“当前到底还在原结构里，还是已进入新结构”的关键。
- 走势具有多义性，但多义性不是随便解释，而是在严格结构约束下存在的若干候选路径。

对 Agent B 的含义：
- 先判断最近三个月日K更像趋势推进、震荡扩张还是中枢延续。
- 不要一看到高点回落就喊背驰；要先确认之前存在同向推进。
- 如果结构存在多解，要列出主解和次解，并说明哪个条件会让主解失效。

核心结论：
- 分型是最小结构识别单元，但不能离开结合律与包含处理孤立使用。
- 笔由相邻有效顶底分型连接形成，顶和底之间至少要满足最基本的K线要求。
- 含有包含关系的K线要先做包含处理，再谈分型与笔。
- 线段至少由三笔构成，但关键不是“数够三笔”，而是结构结合关系是否成立。
- 线段划分必须避免任意性；一旦线段标准被放松，整个缠论分析就会失真。

对 Agent B 的含义：
- 回答时必须优先说明：最近三个月日K里，哪些高低点更像候选顶/底分型。
- 如果分型间包含关系较多、波动噪音较重，要明确说“当前仅能看到候选笔，线段确认度不足”。
- 如果能识别出至少一到两段较清晰波段，应直接指出这些波段的大致起止日期和方向。

核心结论：
- 中枢的重点不是画盒子，而是识别重叠区是否构成稳定结构核心。
- 走势离开中枢后的回抽方式，决定趋势延续还是回归震荡。
- 中枢可以扩张，也可能升级为更大级别中枢。
- 中枢震荡时，内部小级别波动很多，不能随意上升到大级别趋势结论。

对 Agent B 的含义：
- 用户问最近三个月日K时，要尽量指出：
  - 是否存在一个明显价格重叠区
  - 该重叠区大致价格范围
  - 当前价格在中枢上沿、中部还是下沿附近
- 如果只能判断“疑似中枢”，必须注明疑似，不要硬说已经完成。

核心结论：
- 背驰是走势力度衰减，不等于立刻反转。
- 背驰必须说明级别：笔级别、线段级别还是更大级别。
- MACD 可以做辅助，但不是背驰本体。
- 小级别背驰可以引发大级别转折，但这是“可能触发”，不是“必然触发”。

对 Agent B 的含义：
- 若用户要求缠论分析，必须尽量指出：
  - 最近两段同向推进相比，后一段是否更弱
  - 这种更弱体现在哪些方面：高点创新幅度、低点回撤、斜率、波动扩张或成交配合
  - 当前背驰是“候选背驰”还是“较明确背驰”

## 买卖点

对应文章范围：
- 20
- 21
- 26
- 27
- 49
- 53
- 55

核心结论：
- 买卖点必须建立在结构、中枢、背驰之后。
- 一类买卖点偏向趋势切换的初始信号，二类买卖点偏确认后的回抽，三类买卖点偏离开中枢后的延续确认。
- 三类买卖点很重要，但最容易被滥用：不是任何突破回踩都叫三买三卖。

对 Agent B 的含义：
- 不能直接说“这里就是买点”，要说“更接近哪一类买卖点的候选形态”。
- 必须写清楚触发条件，比如：
  - 若回踩不破某低点/某中枢下沿，则更接近二买或三买确认
  - 若重新跌回中枢，则原判断失效

## 同级别分解与多义性

对应文章范围：
- 32-40
- 77
- 86
- 91
- 93
- 99

核心结论：
- 同级别分解是把复杂走势拆成同一层次上的若干段落来分析。
- 多义性是结构现实，不是分析失效；但多义性必须被约束在清晰规则里。
- 一根筋思维是缠论实战大敌：不能一旦看多就只找看多证据，也不能一旦看空就拒绝修正。

对 Agent B 的含义：
- 在最近三个月日K分析里，如果存在两种合理拆分，应明确写：
  - 主解：当前更倾向哪种分解
  - 次解：如果后续出现什么信号，会切换为另一种分解

核心结论：
- 图解类文章的重要价值，不在“背答案”，而在于教会如何把抽象规则落到具体K线。
- 区间套用于在大级别判断背景下，用小级别寻找更精确位置。
- 月线、日线、中阴阶段、底部演化等案例，强调的是“级别联动下的结构观察”，不是单独某根K线。

对 Agent B 的含义：
- 当用户只问最近三个月日K时，也可以补一句：
  - 当前只是日K级别观察，若要做更高置信度判断，最好联立周线或月线背景
- 但不要在用户未要求时，把整套月线大势分析重新写一遍。

  - 如果失效，更可能退化成震荡、还是进入反向结构

## 常见误区

1. 把任意拐点都当分型。
2. 把任意三笔重叠都当成熟中枢。
3. 把任意弱势上涨都当背驰。
4. 把任意回踩都当二买三买。
5. 忽略级别，拿分钟级别结论硬套到日线。
6. 忘记走势多义性，只保留单一叙事。
7. 用户明明只问最近三个月日K，却擅自扩展到整份股票研究报告。

## Agent B 专用输出模板

1. 分析级别与区间
   - 明确这是最近三个月日K，必要时补充月K仅作为背景。
2. 走势概览
   - 给出区间起止、涨跌幅、最高价/最低价及日期、当前相对位置。
3. 分型与笔
   - 指出最近较重要的候选顶底分型和可能形成的笔。
4. 线段与中枢
   - 判断是否已形成较清晰线段，是否存在明显中枢或疑似中枢。
5. 背驰与买卖点
   - 判断是否有候选背驰，以及更接近哪类买卖点。
6. 风险与不确定性
   - 写清楚哪些结构还没走完，哪些结论依赖后续确认。
